Invento v2 is an inventory system created with Codeigniter 4, along with other technologies such as jQuery, Axios, Datatables, and more. With Invento v2, you'll be able to manage your inventory in one or any number of warehouses you have. If you also have workers and supervisors, no problem! Invento v2 includes user roles. Read the documentation below to learn more about this system.

Features

Invento v2 has the following features:

  • Manage your inventory in any number of warehouses
  • Get statistics about your incomes and expenses
  • Manage categories and brands for your items
  • 3 user roles: admin, supervisor and worker
  • Manage your customers and suppliers
  • Create sales and purchases.
  • Create returns for sales and purchases
  • Be able to make quantity adjustments
  • Be able to make transfers between warehouses
  • Translation supported (included by default English and Spanish)
  • Access to a free app for Android with basic functionality, downloadable from the Play Store

Getting started

Invento v2 was created using Codeigniter 4, so in order to be able to install it, you'll need to meet these requirements:

  • Apache*
  • PHP 7.4+
  • intl extension enabled
  • mbstring extension enabled
  • MySQL 5.1+ with the MySQLi driver
  • mod_rewrite enabled
  • PHP extensions: php-json, php-mysqlnd, php-xml

    • * Codeigniter 4 also supports Nginx and Vagrant, but Invento v2 has not been tested on those platforms. Thus we do not assure Invento v2 will work in any of those platforms, only Apache.

Installation (as main page)

Follow these steps if you would like to install Invento v2 as a main page (i.e. https://yourwebsite.com/)

  1. In the zip file you downloaded from CodeCanyon, you'll find a file named Inventov2_v1.0.2_.zip. Extract its contents.
  2. Upload ALL contents extracted, to your server's public folder. It is tipically named public_html or htdocs. Make sure your tree looks like this:

    Fig. 26

  3. Open the inventov2/.env file with your preferred text editor, and set your MySQL credentials (hostname, database, username and password).
  4. Also, set your website's URL (app.baseURL). For example: https://yourwebsite.com/. Save the file.
  5. Finally, visit https://mywebsite.com/install.php, and follow the instructions. Done!

Remember, your login credentials will be admin (username) and 123456 (password). Remember to change your password.

PS - Once the installation is complete, please delete the install.php and install_queries.php files :)

Updating (from any version to 1.0.7)

A new version was released on January 7th, 2022. This version addresses bug fixes and new features, and you should follow these steps to keep your system up to date.

  1. Download the update from CodeCanyon.
  2. Inside the file you downloaded, you'll find a file named Inventov2_v1.0.7_.zip. Extract its contents.
  3. Upload and replace all files in your current Invento v2 installation, except your .env and .htaccess files.
  4. Go to your admin dashboard to apply the update. Done!

Note: Logging into your admin dashboard is very important to apply the update!

Overview

The main idea behind Invento v2, is to be able to manage your inventory in one or several warehouses by logging the entrance and exit of merchandise. The latter is done by submitting purchases and sales.

When you purchase items from one of your suppliers, stock will be added to a specific warehouse. When you sell items to one of your customers, stock will be deducted from a specific warehouse. This way, Invento v2 is able to keep track of all quantities, and to show you basic charts in your admin panel.

Warehouses

As soon as you install Invento v2, your first step should be creating your warehouses, since the entire system will work with them.

In the Warehouses page, you'll find a list of warehouses.

Fig. 1

If you've just installed the system, this table will be empty. To create a new warehouse, click on New warehouse. Then, fill the warehouse details and click on Create warehouse. Create as many warehouses as you have.

Suppliers

Suppliers are people or companies you buy merchandise from. You get stock into any of your warehouses by buying items from suppliers. Thus, in order to submit new purchases, you must have at least one supplier.

In the Suppliers page, you'll find a list of suppliers.

Fig. 2

If you've just installed the system, this table will be empty. To create a new supplier, click on New supplier. Then, fill the supplier details and click on Create supplier. Create as many suppliers as you need.

Customers

Customers are people or companies you sell merchandise to. You get stock out of any of your warehouses by selling items to customers. Thus, in order to submit new sales, you must have at least one customer.

In the Customers page, you'll find a list of customers.

Fig. 3

If you've just installed the system, this table will be empty. To create a new customer, click on New customer. Then, fill the customer details and click on Create customer. Create as many customers as you need.

Brands

In order to organize your items in a better way, you can assign a brand to them. Please note this is totally optional.

If you wish to have brands, you'll find them in the Brands page.

In order to organize your items in a better way, you can assign a category to them. Please note this is totally optional.

If you wish to have categories, you'll find them in the Categories page.

Items

Invento v2 treats items as something global across all of your warehouses. Thus, when you create new items, no warehouse is linked to them: they belong to the entire system.

Go to Items, and click on New item

Fig. 6

This page might look a bit overwhelming at first, but it's very simple and useful. Here's a description of all of the fields. Those marked with an asterisk (*) are required.

  • Item name* - The name of the item. For example: iPhone XS
  • Barcode type - If you wish to use barcodes, select the type of barcode this item will use. If not, leave it at none.
  • Code* - Enter the item's code. If you selected a barcode type, the code must be compliant. If you selected none, you can enter any series of alphanumeric codes.
  • Brand - If you want, you can assign a brand to this item.
  • Category - If you want, you can assign a category to this item.
  • Sale price* - The price of this item when selling it to a customer.
  • Sale tax - Tax (in percent) of this item when selling it to a customer. It is added to the sale price.
  • Description - Any description for this item
  • Dimensions - If you want, you can save the dimensions of this item (weight, width, height and depth)
  • Minimum quantity alert - If you set this value, you will receive an alert if the item reaches this number in any of your warehouses.
  • Maximum quantity alert - If you set this value, you will receive an alert if the item reaches this number in any of your warehouses.
  • Notes - Any notes for this item

After creating this item, you will be redirected to the item page, where you'll be able to assign suppliers for it. Before creating any purchases, you must assign at least one supplier for your item.

Fig. 7

Click on Add supplier

Fig. 8

  • Supplier* - Which supplier you would like to assign to your item
  • Price* - The price that your supplier has for this item
  • Tax - The tax that your supplier has for this item (will be added to the price)
  • Part number - The part number that your supplier uses for this item

Once you save your suppliers, you'll be able to see and manage them from the item page.

Purchases

Whenever you purchase merchandise from one of your suppliers, you have to submit a new purchase. When you do so, items purchased will be added to your stock and accounting. To create a new purchase, go to the Purchases page, and click on New purchase

Fig. 10

Before you start to add items, you must generate a new reference (or assign your own), select a warehouse, and select a supplier. Bear in mind that you'll able to add only items that have a supplier assigned, and quantities purchased will be added to the stock in the particular warehouse you select.

After filling those 3 details, the rest of the inputs will be enabled.

Start adding items by searching them by name or code. Enter the amounts to purchase, and watch the order amounts change automatically.

Set your shipping cost, discount (in amount) or tax (in percent) for this particular order. Also, set notes if needed.

Fig. 11

Once you're happy with all of the details, click on Create purchase. Purchases cannot be deleted, so make sure everything is correct before submitting it.

Once the purchase is submitted, quantities purchased will be added to the inventory of the particular warehouse you selected.

Purchase returns

There are times when you would like to return merchandise you purchased from a supplier, due to several reasons. When this is the case, you have to submit a new purchase return. When you do so, items returned will be subtracted from your stock (for the specific warehouse where the original purchase took place). To create a new return, go to the Purchase returns page, and click on New return

Fig. 12

First, you must enter the original purchase reference, and press enter. The purchase details will be loaded, please confirm everything matches the purchase you were looking for.

Set the quantity to return of the items that will be returned to the supplier.

Set shipping cost, discount (in amount) and tax (in percent) for this return.

Fig. 13

Once you're happy with all of the details, click on Create return. Returns cannot be deleted, so make sure everything is correct before submitting it.

Once the return is submitted, quantities returned will be subtracted from the inventory of the particular warehouse the original purchase took place.

Sales

Whenever you sell merchandise to one of your customers, you have to submit a new sale. When you do so, items sold will be subtracted from your stock and accounting. To create a new sale, go to the Sales page, and click on New sale

Fig. 14

Before you start to add items, you must generate a new reference (or assign your own), select a warehouse, and select a customer. Bear in mind that quantities sold will be subtracted from the stock of the particular warehouse you select, and that you cannot sell more items than what you have in that warehouse.

After filling those 3 details, the rest of the inputs will be enabled.

Start adding items by searching them by name or code. Enter the amounts to sell, and watch the order amounts change automatically.

Set your shipping cost, discount (in amount) or tax (in percent) for this particular order. Also, set notes if needed.

Fig. 15

Once you're happy with all of the details, click on Create sale. Sales cannot be deleted, so make sure everything is correct before submitting it.

Once the sale is submitted, quantities sold will be subtracted from the inventory of the particular warehouse you selected.

Sale returns

There are times when one of your customers would like to return merchandise you sold to them, due to several reasons. When this is the case, you have to submit a new sale return. When you do so, items returned will be added back to your inventory (for the specific warehouse where the original sale took place). To create a new return, go to the Sale returns page, and click on New return

Fig. 16

First, you must enter the original sale reference, and press enter. The sale details will be loaded, please confirm everything matches the sale you were looking for.

Set the quantity to return of the items that the customer will return to you.

Set shipping cost, discount (in amount) and tax (in percent) for this return.

Fig. 17

Once you're happy with all of the details, click on Create return. Returns cannot be deleted, so make sure everything is correct before submitting it.

Once the return is submitted, quantities returned will be added back to the inventory of the particular warehouse the original sale took place.

Quantity adjustments

There are times when you would like to make quantity adjustments in any of your warehouses, without having to submit a sale or purchase (and thus, without affecting your accounting). To create a new quantity adjustment, go to the Quantity adjustments page, and click on New adjustment

Fig. 18

First, select the warehouse where you would like to make the quantity adjustment. Then, start adding the items you would like to adjust.

For each and every item you added, select the adjustment type (add or subtract), and set the amount of items to add/subtract.

Fig. 19

Once you're happy with all of the adjustment details, click on Create adjustment. Adjustments cannot be deleted, so make sure everything is correct before submitting it.

Once the adjustment is submitted, quantities will be updated in the warehouse you selected.

Transfers

There are times when you would like to transfer stock from one warehouse to another, without having to submit sales, purchases or quantity adjustments. To create a new transfer, go to the Transfers page, and click on New transfer

Fig. 31

First, select which warehouses you would like to transfer stock from and to. Then, start adding the items you would like to transfer.

For each and every item you added, set the quantity to transfer. The last two columns in the table will show the stock change to occur in both warehouses.

Fig. 32

Once you're happy with all of the transfer details, click on Create transfer. Transfers cannot be deleted, so make sure everything is correct before submitting.

Once the transfer is submitted, quantities will be updated in the warehouses you selected.

Alerts

When you created a new item, you had the option to set a minimum quantity alert and maximum quantity alert. Whenever you create a sale, purchase, return or quantity adjustment, the system will detect if the quantity of the affected items are outside of the range you set. If it is, a new alert will be generated.

Fig. 20

You can either click the bell in the header, or click the Alerts option in the sidebar.

In the alerts page, you will find a table with all of the alerts generated, along with: the item name, warehouse name, the alert type (minimum or maximum), the setpoint you saved, the current quantity, and the date when the alert triggered.

The only way to remove alerts is by either editing the setpoints of the item, or by adjusting the quantity (with a sale, purchase, return, or quantity adjustment)

User roles

Invento v2 comes with 3 user roles: admin, supervisor and worker.

Admins and supervisors will see the exact same initial dashboard when they log in. However, information will be different: supervisors will only see information about the warehouses you've given them access to.

On the other hand, workers will see the following initial dashboard, where they will have access to several quick actions:

Users

Users can only be managed by administrators. This includes username/email/password changes.

To create a new user, go to the Users page and click on New user

Fig. 22

Enter the information about the new account, and select the desired role. Please consider the following:

  • Roles are permanent, once you create the account, you won't be able to modify the role
  • If you create an administrator account, it will have the exact same privileges as your account, including being able to remove your account.

When you entered all details, click on Create user

You'll be redirected to the details of the user created. If you created a worker or supervisor, you will need to give access to the warehouses you would like. To do so, select a warehouse from the dropdown. Access will be given immediately.

Translating

Invento v2 does support translation, and comes with 2 languages already included: English and Spanish.

Language files are located in the /inventov2/app/Language path.

To create a new language file, duplicate any existing language. For example, if you would like to translate to Chinese (CN), duplicate the /inventov2/app/Language/en directory (along with the Errors.php, Main.php and Validation.php files). Now, open each one of those files with your preferred text editor, and translate line by line.

Done!
